# Differentiator

Программа для работы с математическими выражениями в виде дерева.

## Возможности

* парсинг выражения из файла
* вычисление значения
* дифференцирование
* разложение в ряд Тейлора
* визуализация дерева (Graphviz)
* упрощение выражений

## Входные данные

Выражение читается из файла:

```id="p7a2km"
expression.txt
```

Формат — префиксная запись.

## Сборка

```id="6j8q1w"
make
do
```

## Требования

Для визуализации дерева необходим установленный Graphviz.
